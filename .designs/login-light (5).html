# Handoff: SkillVault — App Screens (Light theme) ## Overview SkillVault is a catalog platform for AI-agent skills (harness-compatible SKILL.md packages). This bundle covers the core authenticated app screens in the **light theme**: Dashboard/login shell, Categories admin, Skills catalog, "Mis Skills" management, and the multi-step "Publicar Skill" wizard. A dark-theme variant of the dashboard is also included for reference. ## About the Design Files The HTML files in this bundle are **design references built as interactive HTML prototypes** — they show intended look, structure, and behavior (including working filters, sorting, a functional multi-step wizard with state, and modals/toasts) but are not production code to copy directly. The task is to **recreate this design in the target codebase's React environment**, following React + the app's existing patterns, organized with **Atomic Design** (atoms → molecules → organisms → templates → pages). ## Fidelity **High-fidelity.** Colors, typography, spacing, and interaction states below are final — implement pixel-accurately. Sample data (skill names, counts, dates) is illustrative placeholder content; wire to real data. ## Screens ### 1. Dashboard (`login.html` / `login-light.html`) Two-panel shell: sidebar (logo, nav, SSO login, version/status) + KPI grid (Skills publicados, Instalaciones de skills, Contribuyentes) + 4 "Top 5" ranked lists (instalaciones, stars, contribuyentes, categorías). See prior handoff section below for full spec. ### 2.
